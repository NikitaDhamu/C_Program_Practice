# C_Program_Practice :-
This repository contains a collection of C programs created for learning and practice purposes. It includes examples covering basic concepts such as variables, loops, functions, arrays, pointers, and file handling. The goal is to build a strong foundation in C programming through hands-on coding exercises and structured practice

  case 3:

    break;
  case 4:

    break;
  case 5:

    break;
  case 6:

    break;
  case 7:
    printf("Sunday");
